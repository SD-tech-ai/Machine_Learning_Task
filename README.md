# House Price Prediction

## Overview

This project predicts house prices using Linear Regression and KNN Regressor.

## Objective

To build and compare models for accurate house price prediction.

## Technologies

Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn

## Steps

* Data preprocessing
* Feature scaling
* Model training
* Model evaluation (MSE, R² score)

## Models

* Linear Regression
* KNN Regressor

## Result

Performance depends on data and tuning. KNN improves with optimal K value.

## Learnings

* Importance of preprocessing
* Effect of feature scaling
* Model comparison
